PROJECTS

R.D.P.O.D insights

The RDPOD project combines distance-based and density-based clustering techniques with large language models (LLMs) for improved outlier detection. It provides a powerful framework for anomaly detection in complex datasets. RDPOD aims to strengthen data integrity and decision-making across diverse applications.

Drowse Defender

Created a drowsiness detection system to enhance driver safety through computer vision, using a 4-layer CNN with convolutional, ReLU, pooling, and fully connected layers. Integrated analysis of eye movements and yawning to identify drowsiness. Triggered alerts if eyes stayed closed for 7 consecutive frames, ensuring timely intervention.

VisualSense-QA

Developed an end-to-end Visual Question Answering (VQA) web application utilizing advanced AI techniques, including the Vision-and-Language Transformer (ViLT) for image understanding and natural language processing. Addressed challenges in large language models (LLMs) related to interpretability.
